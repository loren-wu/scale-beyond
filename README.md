# 尺度之外 / Scale Beyond

《尺度之外 / Scale Beyond》是一个全屏 3D 宇宙尺度探索网页。它不把网页当作传统的滚动页面，而是把浏览器变成一台安静的宇宙摄影机：用户通过滚轮拉近或拉远尺度，通过拖拽移动视角，从星球局部逐渐退到完整地球，再进入地月系统与太阳系。

项目追求真实、克制、深邃的天文纪录片气质，避免赛博霓虹、游戏 HUD、代码雨和过度科幻 UI。核心目标不是堆砌知识点，而是让用户感受到尺度变化本身带来的震撼。

## 当前 MVP

当前版本已完成第一阶段 MVP，对应 `scaleValue` 0-800：

- 全屏 Three.js 场景与 `PerspectiveCamera`
- 滚轮缩放，而不是页面滚动
- 左键拖拽视角，并带有平滑阻尼
- 星球局部层：程序化地球纹理、云层球体、大气 Fresnel 效果
- 完整星球层：地球整体视角、云层缓慢自转、克制光照
- 地月系统 / 太阳系层：太阳、月球、主要行星与轨道线
- 极简 UI：项目名、当前尺度、尺度单位、核心文案、操作提示和缩放进度条

当前地球与行星材质仍是程序化临时方案，后续会替换为授权明确的 NASA 等真实素材。

## 交互方式

- 滚轮向后：拉远镜头，进入更宏观尺度
- 滚轮向前：拉近镜头，返回更具体尺度
- 鼠标左键拖拽：移动观察视角
- 移动端：当前已完成基础适配，后续会继续优化触控缩放体验

## 本地预览

这是一个纯静态网页，不需要构建步骤。可以用任意静态服务器打开项目根目录，例如：

```bash
python -m http.server 4173
```

然后访问：

```text
http://localhost:4173/
```
