# 尺度之外 / Scale Beyond

《尺度之外 / Scale Beyond》是一个全屏 3D 宇宙尺度探索网页。它不把网页当作传统的滚动页面，而是把浏览器变成一台安静的宇宙摄影机：用户通过滚轮拉近或拉远尺度，通过拖拽移动视角，从地球表面逐渐退到地月系统、太阳系、银河系，最终进入本星系群。

项目追求真实、克制、深邃的天文纪录片气质，避免赛博霓虹、游戏 HUD、代码雨和过度科幻 UI。核心目标不是堆砌知识点，而是让用户感受到尺度变化本身带来的震撼。

## 当前版本

当前版本已完成七段连续尺度，对应 `scaleValue` 0-1800：

- 全屏 Three.js 场景与 `PerspectiveCamera`
- 滚轮缩放，而不是页面滚动
- 左键拖拽视角，并带有平滑阻尼
- 星球局部层：NASA Blue Marble 白昼、Black Marble 夜灯、独立云层、昼夜分界与大气散射
- 完整星球层：地球整体视角、海洋高光、高纬极光和独立云层自转
- 地月系统层：独立月球、压缩后的地月距离与过渡轨道
- 太阳系层：八大行星程序细节、主要卫星、土星环带与卡西尼缝、天王星环、小行星带、太阳日冕
- 恒星邻域层：太阳系淡出、NASA 多波段银河中心全景与银河模型连续过渡
- 银河系层：中央棒状核球、四条旋臂、盘面、恒星晕和猎户臂太阳位置标记
- 本星系群层：银河系、仙女座、三角座、大小麦哲伦云及克制的名称标注
- 极简 UI：七段中英文叙事、距离读数、位置提示、操作提示和六段尺度条

Earth、Earth at night、clouds、Moon 和银河中心五张 NASA 影像已经随项目本地化，页面优先读取仓库内资源；本地文件缺失时才尝试 NASA 官方地址，全部失败时继续使用程序化 fallback，避免画面中断。行星表面和三维星系结构为基于 NASA 资料设计的程序化可视化，不冒充严格等比例天文模拟。素材与结构参考见 [`ASSET_SOURCES.md`](ASSET_SOURCES.md)。

## 交互方式

- 滚轮向后：拉远镜头，进入更宏观尺度
- 滚轮向前：拉近镜头，返回更具体尺度
- 鼠标左键拖拽：移动观察视角
- 移动端：单指拖拽、双指缩放
- 键盘：方向键或 Page Up / Page Down 切换尺度，Home / End 返回两端

## 本地预览

这是一个纯静态网页，不需要构建步骤。Three.js 与 NASA 纹理均已随仓库本地化，预览时不依赖第三方 CDN。可以用任意静态服务器打开项目根目录，例如：

```bash
python -m http.server 4173
```

然后访问：

```text
http://localhost:4173/
```
